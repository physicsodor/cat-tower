# cat-tower

캣-타워는 시각적 마인드맵을 기반으로 한 스케쥴러를 목표로 개발한 웹앱입니다.
캣-타워를 활용해서 해야 할 일의 우선순위, 학교 교과과정의 선행과목 등을 정리해보세요.
또는 전공 교재의 장절항목을 요약하여 캣-타워에 저장함으로써 효율적으로 시험공부를 해보세요.

## 주요 기능

- **할 일의 우선순위**: 항목 간 순서를 화살표로 시각화
- **할 일의 그룹화**: 여러 항목을 그룹화하여 관리
- **수식 지원**: Latex 문법을 통해 수식이 포함된 내용 작성
- **무한 캔버스**: 무한히 펼쳐진 공간에서 항목을 자유롭게 배치·이동하고 탐색
- **프로젝트 저장 및 공유**: 클라우드 저장/불러오기 및 url 공유

## 기술 스택

- **React 19** + **TypeScript**
- **Vite** (빌드 도구)
- **Supabase** (인증 + DB)
- **KaTeX** (수식 렌더링)
- **Sass** (스타일)
- **fractional-indexing** (순서 관리)
- **lz-string** (데이터 압축)

## 로컬 실행

```bash
npm install
npm run dev
```

빌드:

```bash
npm run build
```
